## 📚 Sobre o Projeto
Este arquivo HTML marca meus estudos sobre **elementos semânticos** introduzidos no HTML5. Aprendi a diferença entre tags semânticas (que dão significado ao conteúdo) e tags genéricas como `<div>` e `<span>`, além de explorar estilização inline com CSS.

## 📁 Arquivo
| Arquivo | Descrição |
|---------|-----------|
| `semantico.html` | Demonstra o uso de elementos semânticos como `<header>`, `<nav>`, `<main>`, além de `<div>` e `<span>` |

## 🎯 O que aprendi

### Tags Semânticas Utilizadas:

| Tag | Significado | Função |
|-----|-------------|--------|
| `<header>` | Cabeçalho | Representa o cabeçalho da página ou de uma seção |
| `<nav>` | Navegação | Agrupa links de navegação principais |
| `<main>` | Conteúdo principal | Contém o conteúdo central da página |

### Tags Genéricas (não semânticas):

| Tag | Significado | Função |
|-----|-------------|--------|
| `<div>` | Divisão | Container genérico em bloco |
| `<span>` | Span | Container genérico em linha |

## 🖥️ Exemplos do Código

### Estrutura Semântica Básica
```html
<header>
    <nav>
        <a href="empresa.html">A empresa</a> |
        <a href="produtos.html">Produtos</a> |
        <a href="fale.html">Fale conosco</a> |
    </nav>
</header>

<main>
    <p>Conteúdo principal do meu site é definido pelo main</p>
</main>

<body>
    <header>
        <nav>
            <!-- Menu de navegação -->
        </nav>
    </header>
    
    <main>
        <!-- Conteúdo principal -->
        <article>
            <!-- Artigo independente -->
        </article>
        <section>
            <!-- Seção do conteúdo -->
        </section>
    </main>
    
    <aside>
        <!-- Conteúdo complementar (sidebar) -->
    </aside>
    
    <footer>
        <!-- Rodapé -->
    </footer>
</body>

┌─────────────────────┐

│  ┌─────────────────┐│
│  │     <nav>       ││
│  │  Links de menu  ││
│  └─────────────────┘│
├─────────────────────┤
│                     │
│      <main>         │
│  Conteúdo principal │
│                     │
├─────────────────────┤
│      <footer>       │
│  Rodapé da página   │
└─────────────────────┘
